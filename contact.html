Frequently Asked Questions

Is PDF Tools Play really free?

Yes! PDF Tools Play is completely free with no hidden charges, watermarks, or registration required. We believe in providing accessible tools for everyone.

Are my files safe and private?

Absolutely! All processing happens locally in your browser. Your files never leave your device or get uploaded to any server. This ensures maximum privacy and security.

What file formats are supported?

We support JPG, PNG, and WEBP for images, and PDF files for merging and collage creation. Files are limited to reasonable sizes to ensure smooth processing.

Can I use PDF Tools Play on mobile?

Yes! Our tools are fully responsive and work on all modern browsers including mobile devices. The experience is optimized for both desktop and mobile use.

Do you offer technical support?

While we don't offer live technical support, we respond to all emails within 24-48 hours. For common issues, check our FAQ section or try clearing your browser cache.

Before Contacting Us

For faster resolution, please check these common solutions:

Clear Browser Cache

If tools aren't loading properly, try clearing your browser cache and reloading the page.

Try Another Browser

Some browser extensions may interfere. Try using a different browser or disabling extensions.

Check File Size

Large files may take longer to process. Check file size limits on each tool's page.

Check Internet Connection

While processing is local, initial page load requires internet for libraries.
